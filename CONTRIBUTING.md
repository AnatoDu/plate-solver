# Участие в разработке

`plate-solver` — научный комплекс программ (изгиб и односторонний контакт
упругих пластин произвольного очертания). Развивается преимущественно одним
автором как исследовательский продукт, но замечания, сообщения об ошибках и
предложения приветствуются.

## Сообщить об ошибке или предложить улучшение

Открывайте issue: <https://github.com/AnatoDu/plate-solver/issues>. Для ошибки
приложите минимальный воспроизводимый пример (case-файл или короткий скрипт),
ожидаемое и фактическое поведение, версию пакета (`plate-solve --version`).

## Окружение разработчика

```bash
python -m venv .venv && source .venv/bin/activate
pip install -e ".[dev]"          # + ".[dev,fem]" для независимой МКЭ-верификации
```

## Перед предложением изменений

- **Тесты зелёные:** `pytest -m "not big and not fem"` (быстрые); полный — `pytest`.
- **Стиль:** `ruff check .` (line-length 100).
- **Каждый численный метод** сопровождается тестом и кратким математическим
  обоснованием (порядок аппроксимации, предполагаемая оценка погрешности).
- **Не ослаблять ворота верификации** ради прохождения: сверка 1D↔2D и
  редукционная лестница — сертификаты корректности метода; при расхождении
  причину ищут в методе и краевых условиях, а не в допуске.
- Эталонные отчёты (`results/reference/`) под SHA-256-воротами: изменение чисел —
  осознанным коммитом с обоснованием в CHANGELOG, а не тихой правкой.
- Публичные функции — с docstring; математику пояснять формулой; документация —
  на языке продукта.
