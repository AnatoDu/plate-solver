## Фаза 3 — трек A (контактные расширения)

- A1 (поле зазора): ContactMOR принимает скаляр ИЛИ массив в узлах
  (скалярный путь побитово прежний — золотые ворота не сдвинулись);
  схема [contact.gap] kind=const|plane|paraboloid|steps (ровно одно из
  gap | gap_factor | таблица); const → скаляр (тождество т1 побитово).
  Ворота: т1 побитово; т2 (plane ≥ 2·w_free ⇒ r≡0); т3 steps-зона
  value=10³·w_free ≡ исключению зоны из маски (≤1e-12); т4 paraboloid:
  снимок r_curv=1e4 (r_max=62.14, 68 узлов) + сходимость к плоскому
  3.117e-4 → 2.666e-5 → 2.666e-6 (монотонно; потолок 1e-3 пройден,
  заморожено факт×3 = 8.0e-6). Метрики комплементарности у поля
  нормируются min Δ на основании.

- A2 (силовой штамп): схема contact.force (>0); при force скалярный
  gap/gap_factor игнорируется (warning), [contact.gap] — форма штампа
  (Δ = level + shape). Реализация: F(level)=∫r dΩ−P монотонно убывает,
  brentq на [касание, отрыв], каждый вызов — полный МОР с тёплым стартом
  r0 (новый параметр ContactMOR.solve; дефолт — прежний холодный путь).
  **Диагностика:** недосошедший МОР смещает F — первый прогон ворот
  (max_iter=2000) дал 3.7e-3 вместо потолка 1e-6; на ГЛУБОКОЙ сходимости
  (tol=1e-11 достигается: 251k итераций, 24 с на Q=40) замыкание
  позиционный↔силовой: |Δlevel|/w_free = 2.253e-10 (потолок 1e-6),
  ∫r rel = 1.063e-10 (потолок 1e-8) → заморожено факт×3 = 7e-10 / 3.2e-10.
  Экономия тёплого старта: 174954 итер./вызов против 251537 холодного
  (~30 % — глубокий dr-хвост кромки проходится в любом случае).
  Снимок ladder lshape_stamp_force (P=0.5, протокол golden): level=4.766e-5,
  r_max=783.6, 13 узлов, ∫r=0.50039 (протокольное недосхождение ~8e-4 —
  зафиксировано в baselines с пояснением); предельный P→0⁺ ⇒ level→w_free;
  P сверх касания — CaseError с диапазоном.

- A3 (контактный интерфейс защемления): ClampedPlate.solve_rhs —
  факторизация (диаг. предобуславливание + Холецкий) один раз в __init__,
  прежний solve — псевдоним (т1 побитово); deflection_at_quad /
  laplacian_at_quad — кэши ψ и Δψ (Δψ уже считался при сборке S — просто
  сохранён, резидентная память не выросла против пика сборки). Общий
  протокол контакта у обеих плит: w_at_quad / lap_w_at_quad / coeffs_w;
  ContactMOR переведён на протокол (PlateBending-путь бит-в-бит —
  золотые ворота на месте). Запреты contact+clamped и ktn+clamped СНЯТЫ
  (валидатор, диспетчер; КТН-кривизна защемления — из кэша Δ(ω²Φ)).
  **Диагностика т2:** ворота Δw сформулированы по машинерии (коэффициенты
  точного решения заданы аналитически, Φ≡const): rel ≤ 1e-10 ✓; гейтить
  Δw РЕШАТЕЛЯ на круге нельзя — квадратурный пол маски ~1/Q усиливается
  второй производной до ~19 % при Q=128 (зафиксировано как свойство,
  не дефект: у прогиба тот же пол даёт ~0.4 %).
  A3.4: предельный gap huge ⇒ r≡0 и изгиб (1e-10) ✓; инварианты ✓;
  комплементарность факт 3.123e-2 (потолок 5e-2) → заморожено 9.4e-2;
  overshoot 2.805e-3 → 8.5e-3. Case circle_clamped_contact (ladder Q=256
  + ci): снимок r_max=50.76, 332 узла, w_max=7.5004e-4. Инфо: пик
  защемления НИЖЕ шарнирного (44.52 против 60.95 на лёгкой постановке) —
  материал главы 4.

- A4 (пара пластин): схема contact.target=plate2 + [plate2] (bc/load
  обязательны, прочее — от первой; gap=0 — касание; force и ktn для пары —
  фаза 5). TwoPlateMOR: узлы контакта — квадратура первой ∩ ω₂>0;
  межсеточного переноса нет (w₂ в узлах первой — прямое вычисление
  структуры ψ₂(x₁); вклад r в нагрузку второй — b₂ += ψ₂(x₁)·(W₁·r) по
  родной квадратуре r); gain = gain₁+gain₂ (теорема 4); низкоуровневые
  входы solve_b/load_vector/solve_from_b/structure_at у обеих плит
  (smoke: solve_from_b(load_vector(f)) ≡ solve(f)).
  **Отступление (a):** узловая реакция при полном контакте НЕединственна
  (стационарность фиксирует только проекцию ψᵀW(q₁−q₂−2r)=0, M ≫ N;
  константа не лежит в достижимом span траектории) — континуальные
  потолки (a) гейтятся на НЕПОДВИЖНОЙ ТОЧКЕ: r ≡ (q₁−q₂)/2 стартом →
  остаётся на месте (std/mean = 2.9e-15 ≤ 1e-8, mean точно, w₁≡w₂
  2.1e-15 ≤ 1e-10 — знаки/сборка верны); холодный старт: w₁≡w₂ ≤ 5e-6,
  равнодействующая в 15 %; обратный порядок q₁<q₂ ⇒ r≡0 точно.
  **Нормировка метрик пары:** физический масштаб щели
  (max|w₁_free|+max|w₂_free| от фактических нагрузок; gain-нормировка
  занижала масштаб при q0>1 и давала ложные 5.4e-2>потолка) — β_eff
  отдельно, по теореме 4 от единичной. (b) смешанные закрепления:
  инварианты ✓, comp факт 1.730e-2 (потолок 5e-2) → заморожено 5.2e-2;
  вырождение E₂=1e9·E₁ ⇒ совпадение с основанием gap=Δ: факт ~1.0e-9
  (потолок 1e-6) → заморожено 3.1e-9. (c) круг над кольцом: r≡0 над
