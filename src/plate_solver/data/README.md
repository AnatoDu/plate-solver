# data — эталонные данные пакета

## stamp_maple_xy.txt

Аналитическое решение 1D контактной задачи со штампом (полоса-пластина на
`[0, 100]`, жёсткий штамп с зазором), экспортированное из Maple; использовано
как эталон в публикации: Ермоленко, Дуркин (2022). Формат: две строки CSV —
координаты `x` (целые 0…100) и прогибы `w` в тех же узлах.

Известный дефект исходного экспорта: значение с индексом 45 битое (в файле
стоит «25»). Оно НЕ исправлено в самом файле (данные хранятся как есть,
провенанс важнее красоты) — починка выполняется при загрузке усреднением
соседей, `ya[45] = (ya[44] + ya[46]) / 2`, см.
`plate_solver.stamp.load_maple_reference`.

Файл распространяется вместе с пакетом (`[tool.setuptools.package-data]`
в pyproject.toml); путь к нему разрешается относительно модуля `stamp.py`.
