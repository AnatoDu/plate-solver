# Ноутбуки — обучающая цепочка

Восемь ноутбуков, читаемых подряд: каждый опирается на предыдущий и добавляет
один новый пласт. Вместе они учат пользоваться комплексом и показывают разные
типы задач — формы, нагрузки, теории и контакт — с нарастанием сложности.

## Запуск

```bash
pip install -e ".[notebooks]"   # jupyter + ipympl (для интерактивной 3D-поверхности в 02)
jupyter lab notebooks/
```

Ноутбуки исполнены и закоммичены с выходами (кроме интерактивной ipympl-ячейки
в 02, которая осмысленна только в живом ядре). Метаданные ядра — стандартный
`python3`.

## Как читать

- **01–04** — основы без контакта: от API к case-файлам, композиции области и
  типам нагрузки; всё на классической теории Кирхгофа.
- **05** — первый контакт (классический, две пластины).
- **06** — четыре теории одним ключом `[model] theory`; где и как проявляется
  каждая поправка.
- **07–08** — единая параметрическая модель, морфинг теорий и нелинейный контакт
  методом обобщённой реакции поверх КТН (штамп, взаимный контакт пары).
