# Теория (парадный вход)

Краткое замкнутое изложение того, что считает пакет и почему это
корректно. Тонкости, калибровки и подводные камни — журнал
[NOTES.md](NOTES.md) (ссылки §N ниже — на него).

## 1. Постановка изгиба

Тонкая упругая пластина занимает область Ω ⊂ ℝ² произвольного очертания.
Классическая модель (Кирхгоф):

    D·Δ²w = q̃(x, y),   D = E h³ / (12(1 − ν²)),

w — прогиб (положителен ВНИЗ, как и нагрузка q; ось z — вниз, §0/§19).
Краевые условия: «мягкий шарнир» (w = 0, Δw = 0), жёсткое защемление
(w = 0, ∂w/∂n = 0), истинный шарнир (w = 0, M_n = 0), свободный край
(M_n = 0, V_n = 0 — обобщённая перерезывающая Кирхгофа); на
прямоугольнике типы назначаются сторонам независимо (`bc.mixed`).

## 2. Геометрия R-функциями (метод Рвачёва)

Граница области задаётся одной функцией ω(x, y): ω > 0 внутри, ω = 0 на
∂Ω. Примитивы (круг, полуплоскость, прямоугольник) соединяются
R-операциями системы R0:

    x ∧ y = x + y − √(x² + y²),   x ∨ y = x + y + √(x² + y²),

что даёт нормализованную ω для пересечений/объединений/разностей —
включая невыпуклые области, входящие углы и вырезы. СТРУКТУРА РЕШЕНИЯ
w = ω·Φ (мягкий шарнир: w|∂Ω = 0) или w = ω²·Φ (защемление: и нормальная
производная гаснет) удовлетворяет кинематическим условиям ТОЖДЕСТВЕННО
при любом Φ; Φ раскладывается по полиномам Чебышёва.

## 3. Численный метод

**Мягкий шарнир** — расщепление бигармоники на две задачи Пуассона
(мембранная аналогия Маркуса): −ΔM = q̃, M|∂Ω = 0; −Δw = M/D, w|∂Ω = 0.
Каждая решается Ритцем на структуре ω·Φ; матрица Ритца факторизуется
один раз (Холецкий), итерации контакта стоят 4 GEMV (§2, §3).
На КРИВОЙ границе мягкий шарнир — самостоятельная модель: от истинного
шарнира он отличается конечным модельным зазором (парадокс
Сапонджяна–Бабушки, §8–9), от которого пакет честно отчитывается.

**Защемление** — прямой Ритц по энергии (D/2)∫(Δw)² на структуре ω²Φ:
эйлерово уравнение — истинная бигармоника, парадокса нет (§9).

**Смешанные КУ и свободный край** (прямоугольник) — полная билинейная
форма

    a(w, v) = D ∬ [ Δw·Δv − (1−ν)(w_xx v_yy + w_yy v_xx − 2 w_xy v_xy) ],

структура (∏ω_c²)(∏ω_h)·Φ по закреплённым сторонам; свободная сторона
множителя не даёт — M_n = 0 и V_n = 0 выходят естественно (§20).
Кинематика обязана исключать жёсткие смещения: ≥ 1 защемлённая либо
≥ 2 шарнирные стороны (ядро {1, x, y}).

**Аппроксимация.** Базис спектральный: на гладких решениях сходимость
экспоненциальная по p; пол точности задаёт квадратурная маска кривой
границы ~O(1/Q) (§13–14) и разрывы нагрузки; логарифмические решения
(точечная сила) приближаются алгебраически (§18).

## 4. Односторонний контакт: метод обобщённой реакции (МОР)

Контакт с жёстким основанием на зазоре Δ — задача Синьорини:

    w ≤ Δ,   r ≥ 0,   r·(w − Δ) = 0   в зоне основания.

МОР — итерация по реакции:

    r ← [ r + β_eff·(u(r) − Δ) ]₊ ,   β_eff = β / gain,

где u(r) — прогиб от нагрузки q − r (классика: u = w; уточнённая
теория: u — прогиб нижней лицевой, §21), gain — максимум прогиба от
единичной нагрузки (нормировка оператора). Сходимость — сжатие при
0 < β < 2 (теорема 4); критерии останова: ‖Δr‖ (`stop="dr"`) или
безразмерная KKT-невязка комплементарности (`stop="comp"`).
Свободная граница (кромка зоны контакта) ищется самим методом; замкнутые
решения существуют лишь при конечномерной свободной границе — 1D и
осесимметрия (оба служат эталонами, включая замкнутый контакт
круг+основание с кольцевой реакцией Кирхгофа на границе зоны).

**Пара пластин**: узлы контакта — квадратура первой пластины на
пересечении планформ; итерация на разности прогибов r ← [r + β((w₁ − w₂)
− Δ)]₊, нагрузки q₁ − r и q₂ + r; межсеточного переноса нет (прогиб
второй вычисляется в узлах первой её же структурой). Контактное условие
пары в v0.3 — классическое (по срединным плоскостям).

## 5. Лестница теорий и уточнённая теория (тип Кармана–Тимошенко–Нагди)

**Честные термины (важно для разграничения).** Тип модели выбирается ключом
`[model] theory` (лестница одним ключом, с ЯВНЫМИ именами):

* `classic` — линейная теория Кирхгофа (расщепление бигармоники);
* `karman` — **геометрически-НЕЛИНЕЙНОЕ** решение Фёппля–Кармана (раздел 5-бис):
  прогиб входит квадратично в мембранные деформации, мембранное поле `N`
  ужестчает пластину;
* `ktn_linear` — **ЛИНЕЙНЫЕ** поправки поперечного сдвига/обжатия (часть теории
  Тимошенко–Нагди) ПОСТОБРАБОТКОЙ на решении Кирхгофа. Это НЕ нелинейная
  теория (прежнее поведение `ktn`, число-в-число);
* `ktn_full` — **ПОЛНАЯ нелинейная КТН** (раздел 5-тер): Карман + оператор
  `(I − h_ψ²Δ)L(Φ, w)` + нагрузочный член `−h_*²Δq_n`.

Устаревшее значение `ktn` неоднозначно и переименовано в `ktn_linear`
(депрекация-алиас, `docs/MIGRATION.md`).

Поправки порядка (h/L)²: поперечный сдвиг (длина h_Ψ² = h²/(6(1−ν))) и
поперечное обжатие. В контактном условии зазор
проверяется прогибом КОНТАКТИРУЮЩЕЙ (нижней) лицевой поверхности:

    u_c = w + (2h_*² − h_Ψ²)·Δw − A·q⁺ − B·D·r,

(коэффициенты — ktn.py; первоисточник — 1D-теория школы; канон пакета —
§21.1). Кривизнный член сглаживает контактную реакцию у кромок штампа и
входящих углов — уточнённая теория не терпит точечных концентраций ни в
нагрузке, ни в реакции (§18).

## 5-бис. Теория Кармана (геометрическая нелинейность, v0.4.0)

Неизвестные — прогиб `w` и перемещения срединной плоскости `u, v`. Мембранные
деформации содержат КВАДРАТИЧНЫЕ по прогибу члены:

    ε_x = u_x + ½w_x²,   ε_y = v_y + ½w_y²,   γ_xy = u_y + v_x + w_x w_y,

усилия (плоское напряжённое, C = Eh/(1−ν²)):

    N_x = C(ε_x + νε_y),   N_y = C(ε_y + νε_x),   N_xy = ½(1−ν)C·γ_xy.

Система Фёппля–Кармана — плоское равновесие плюс внеплоскостное уравнение:

    N_x,x + N_xy,y = 0,   N_xy,x + N_y,y = 0,
    D·Δ²w = q + (N_x w_xx + 2 N_xy w_xy + N_y w_yy).

**Предел N → 0** возвращает классику `D·Δ²w = q` (контроль — Gate L: при w/h ≪ 1
все три теории совпадают с Кирхгофом с точностью дискретизации).

**Требование неподвижной кромки.** Мембранное ужесточение существенно лишь при
`inplane_bc = immovable` (u = v = 0 на ∂Ω): втягивание кромки снимает натяжение,
и эффект почти пропадает (проверено 1D). Основной режим — `immovable`; `movable`
(N·n = 0, полностью свободная кромка) даёт слабый, но НЕнулевой эффект.

**Метод (membrane.py).** Итерация Пикара по ЗАМОРОЖЕННЫМ усилиям `N_k`
(§5.1 постановки): (1) по `w_k` вычислить нелинейные части деформаций;
(2) решить линейную плоскую задачу для `(u, v)` → `N_k`; (3) решить
`(D·S_bend + K_geo(N_k)) c = b` с симметричной геометрической жёсткостью
`K_geo[i,j] = ∫ N_αβ ψ_{i,α} ψ_{j,β}`; (4) недорелаксация; (5) останов по
относительной норме. Большие прогибы (мембранный предел Hencky, w/h ~ 6)
достигаются НАРАЩИВАНИЕМ нагрузки за `n_load_steps` шагов с тёплым стартом;
итерация ускоряется смешиванием Андерсона (та же неподвижная точка, сверхлинейная
сходимость). Мат. обоснование сходимости — кандидат в теорему T5 (NOTES.md):
отображение Пикара — сжатие при нагрузке ниже порога (нелинейность квадратична,
шаги по нагрузке продолжают решение по параметру).

Валидация — комплекс НЕЗАВИСИМЫХ эталонов разной математической природы
(раздел 7): точное решение мембраны Hencky, ряды Way, энергия Ритца Тимошенко,
ряды Фурье Levy, кросс-чек IGA. Числа — `benchmarks.py`.

## 5-тер. Полная нелинейная КТН (`ktn_full`, v0.5.0)

**Иерархия теорий** (операторные формы):

    Кирхгоф: D·Δ²w = q_n
    Карман:  D·Δ²w = q_n + L(Φ, w),               (1/Eh)Δ²Φ = −½ L(w, w)
    КТН:     D·Δ²w = q_n − h_*²Δq_n + (I − h_ψ²Δ)L(Φ, w)

где L(Φ, w) = Φ_,11 w_,22 + Φ_,22 w_,11 − 2Φ_,12 w_,12 = N_x w_,xx + 2N_xy w_,xy
+ N_y w_,yy (при N_x = Φ_,22, N_y = Φ_,11, N_xy = −Φ_,12).

**Параметры толщины** (канонические имена, `faces.py`):

    h_ψ² = h²/[6(1−ν)],  h_*² = (4−3ν)h²/[24(1−ν)],  h_c² = νh²/[8(1−ν)] = h_ψ² − h_*².

При ν = 0.3: h_ψ² = 0.2381h², h_*² = 0.1845h², h_c² = 0.0536h². Все КТН-поправки
порядка O(h²/L²) — при h/L → 0 гаснут (Gate R4).

**Добавочные члены относительно Кармана.** (A) нагрузочный −h_*²Δq_n
(под равномерной нагрузкой Δq_n = 0 ⇒ нужны неравномерные нагрузки — раздел 5-бис
про gaussian); (B) регуляризация связи −h_ψ²ΔL(Φ, w) — главный новый оператор.

**Слабая форма (B) без производных 4-го порядка** (ключевой приём). В форме
Галёркина по формуле Грина `−h_ψ²∫v ΔL = −h_ψ²∫L Δv − h_ψ²∮(v∂_nL − L∂_nv)`.
Для структуры ЗАЩЕМЛЕНИЯ (w = ω²Φ) одновременно v = 0 и ∂_n v = 0 на ∂Ω, поэтому
граничный интеграл ЗАНУЛЯЕТСЯ тождественно — лапласиан переносится с решения на
пробную функцию (вторые производные структуры уже есть). Для мягкого шарнира
остаётся член `+h_ψ²∮L ∂_n v` по полудеформационным условиям — задел (в v0.5.0
КТН реализована на защемлении).

**Вырождения** (основа верификации, редукционная лестница §9, теорема T6):
h_ψ² = h_*² = 0 ⟹ КТН → Карман (Gate R1, машинная точность); тонкая пластина +
малый прогиб ⟹ Карман → Кирхгоф (Gate R2); линейный `ktn_linear` — малопрогибный
предел `ktn_full` (Gate R5).

**Поле поперечного сдвига ψ** (§3.6) — ОДНОСТОРОННЯЯ постобработка: правая часть
уравнения `ψ_1,1 + ψ_2,2 = −(q_n + L)/(μh)` зависит только от найденного (w, Φ),
обратной связи ψ → w нет. Ядро (w, лицевой прогиб, изгибные/мембранные лицевые
напряжения) от ψ НЕ зависит; ψ нужна только для распределения поперечных
касательных напряжений (в v0.5.0 отложена, `shear_output=False`).

Реализация: `ktn_full.KTNPlate` (слой поверх `KarmanPlate`); мат. обоснование —
теорема T6 (NOTES.md).

## 5-quater. Единая параметрическая модель и атлас теорий (v0.6.0)

Архитектурный стержень версии: четыре теории — не отдельные решатели, а
именованные ПРЕСЕТЫ параметров ОДНОГО решателя полной КТН (`ktn_solver.KTNSolver`,
параметры — `theory.TheoryParams`). Редукции точны ПО ПОСТРОЕНИЮ (тот же путь
исполнения), а не «совпадением до допуска» — усиление теоремы T6. Две
управляющие оси задают решётку 2×2:

|                   | уточнение ВЫКЛ (`h_ψ²=h_*²=0`) | уточнение ВКЛ (физические `h_ψ², h_*²`) |
|-------------------|--------------------------------|-----------------------------------------|
| нелинейность ВЫКЛ (`membrane=False`) | `classic` (Кирхгоф)     | `ktn_linear` (сдвиг/обжатие постобработкой) |
| нелинейность ВКЛ (`membrane=True`)   | `karman` (Фёппль–Карман) | `ktn_full` (полная КТН)                    |

**Атлас различий** (что добавляет теория, где видно, коэффициент лицевой кривизны
для контакта `c = h_c²−h_*²`):

| Теория       | Добавочный член (решение) | Где виден в решении        | `solve_ktn_terms` | `face_curv_coeff` (контакт) |
|--------------|---------------------------|----------------------------|:-----------------:|:---------------------------:|
| `classic`    | —                         | линейная жёсткость          | нет               | 0 (контакт по срединной)    |
| `ktn_linear` | — (постобработка)         | ТОЛЬКО лицевые величины      | нет               | `h_c²−h_*²` ≠ 0 (лицевая)    |
| `karman`     | мембранная связь `N:∇∇w`  | «нагрузка–прогиб» вниз       | нет               | 0 (контакт по срединной)    |
| `ktn_full`   | `(I−h_ψ²Δ)L − h_*²Δq`     | срединный (неравн.) + лицевой | да              | `h_c²−h_*²` ≠ 0 (подпись КТН)|

**Морфинг** (`TheoryParams.with_refinement_scale(α)`): непрерывное масштабирование
уточнения `h_ψ²→α h_ψ²`, `h_*²→α h_*²`. При α=1 — полная КТН, при α=0 — Карман
(КТН-члены в решении сняты, `solve_ktn_terms=False`), плавно между — «одна теория
с крутилкой». Концы морфинга ТОЧНО совпадают с пресетами (машинный ноль). Даёт
графики «решение vs α» и демонстрирует коммутативность редукций.

Демонстрация — `notebooks/07_theory_ladder.ipynb` (решётка 2×2, морфинг,
нелинейный контакт, диагностика). Реализация: `theory.py`, `ktn_solver.py`.

## 6. Лицевые поверхности

**Напряжения** (§19, канон Ермоленко–Турковой, b = 0):

    σ_ii(±h/2) = ± 6M_ii/h² + ν/(1−ν)·q_n^±,   σ_12(±h/2) = ± 6M_12/h²,

где q_n⁺ — внешняя нагрузка на верхней лицевой, q_n⁻ — контактная
реакция r на нижней. Знаки: в пролёте при q > 0 низ растянут.

**Прогиб контактирующей лицевой** — модельные формулы пакета
(кинематика КТН, §21.1): w_bot = u_c (см. раздел 5), w_top ≡ w
(канон выделяет формулой только контактирующую поверхность),
dh = w_bot − w — каноническое смещение контактирующей лицевой
(в зоне контакта dh < 0). Поля пишутся в fields.npz (`w_top`, `w_bot`,
`dh`) при `theory = "ktn"`.

Существует и классическое 3D-восстановление лицевых прогибов
интегрированием обжатия ε_z по толщине (§21.2): его кривизный блок
тождественен кинематике КТН (что независимо подтверждает поверхность и
знаки), а различие q,r-членов лежит в пределах O(h²)-точности теории
пластин; в пакете оно служит диагностикой, не каноном.

## 7. Верификация (роли эталонов)

| Пояс | Инструмент | Что даёт |
|---|---|---|
| Замкнутая аналитика | `analytic`, фабрика `analytic_auto` с самосертификацией (PDE + КУ + остаток ряда) | абсолютный эталон канонических постановок |
| MMS | изготовленное решение | сертификат метода на произвольной области |
| Независимый МКЭ | scikit-fem: Морли (Кирхгоф), Аргирис, P2-Marcus | независимая дискретизация, включая свободные края |
| 1D ↔ 2D | 1D-Ритц по радиусу | точка отсчёта без погрешности маски границы |
| Контакт | Maple-эталон 1D-штампа; замкнутый осесимметричный контакт | сходимость МОР к точному решению |
| Карман (большой прогиб) | Hencky (мембрана), Way (ряды), Тимошенко (Ритц), Levy (ряды Фурье), IGA — `benchmarks.py` | невырожденная валидация нелинейного решателя эталонами РАЗНОЙ природы (Gate L/M/K/B) |

Допуски ворот калибруются протоколом «потолок задан → факт × 3» с
обязательным диагнозом природы ошибки (комментарий в case-файле).
