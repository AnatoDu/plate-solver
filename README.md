# plate-solver

Комплекс программ для расчёта **изгиба и одностороннего контакта упругих
пластин произвольного очертания**. Защищаемый результат кандидатской
диссертации (специальность 1.2.2 «Математическое моделирование, численные
методы и комплексы программ», физико-математические науки).

Триада «Модель — Алгоритм — Программа» (по Самарскому):

- **Модель** — уточнённая нелинейная теория типа Кармана–Тимошенко–Нагди
  (поперечный сдвиг + поперечное обжатие);
- **Геометрия** — задание границы произвольной области R-функциями
  В. Л. Рвачёва (ω(x, y) = 0) — *способ задания границы*;
- **Численный метод** — решение бигармоники/Пуассона на произвольной области;
- **Контакт** — метод обобщённой реакции (МОР) для свободной границы.

## Установка

```bash
python -m venv .venv && source .venv/bin/activate
pip install -e ".[dev,examples]"
```

## Запуск

```bash
pytest                          # тесты (включая верификацию 1D↔2D)
python examples/circular_plate.py   # пример: прогиб круглой пластины
ruff check .                    # стиль
```

`pytest` работает и без установки пакета: путь к `src/` прописан в
`pyproject.toml`.

## Структура

```
src/plate_solver/
├── model/      соотношения КТН, постановки задач
├── geometry/   R-функции (граница ω(x,y)=0, операции R0)
├── solver/     бигармоника / Пуассон на произвольной области
├── contact/    метод обобщённой реакции (МОР)
└── analytic/   1D-решения круглой пластины (верификатор)
tests/          автотесты, в т. ч. верификация 1D↔2D
examples/       минимальные воспроизводимые примеры
```

## Статус

Каркас. Уже работают: модель материала, R-функции (примитивы и операции),
аналитические решения круглой пластины, их тесты. Заготовки (с описанием
интерфейса): решатели Пуассона/бигармоники и МОР — наполняются по главам 2–3.
Тест верификации 1D↔2D подключён и автоматически активируется, как только
будет реализован 2D-решатель.
